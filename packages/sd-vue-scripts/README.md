# SD Vue Scripts

A collection of commands used in the SD Design Vue component library

## Installation

```
npm install @sd-design/sd-vue-scripts
```

## Commands

Run `sd-vue-scripts --help` to see all available commands and options.

- `sd-vue-scripts icongen`: generate icon components.
- `sd-vue-scripts lessgen`: generate index less file.
- `sd-vue-scripts dtsgen`: emit .d.ts files for vue files.
- `sd-vue-scripts dev:component`: build components with watch mode.
- `sd-vue-scripts build:component`: build production files.
- `sd-vue-scripts build:style`: build style related files.
- `sd-vue-scripts build:material`: build vue material.
- `sd-vue-scripts test`: run test for component or material.
