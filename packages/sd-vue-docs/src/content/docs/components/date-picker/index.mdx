## API

### `Common` Props

| 参数名 | 描述 | 类型 | 默认值 | 版本 |
| --- | --- | --- | :-: | :-- |
| locale | 国际化配置，用于覆盖locale中的 `datePicker` 字段 | `Record<string, any>` | `-` |  |
| hide-trigger | 没有触发元素，只显示选择面板 | `boolean` | `false` |  |
| allow-clear | 是否允许清除 | `boolean` | `true` |  |
| readonly | 是否为只读 | `boolean` | `false` |  |
| error | 是否为错误状态 | `boolean` | `false` |  |
