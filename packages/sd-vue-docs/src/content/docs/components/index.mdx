## 通用
- [按钮 Button](/components/button/) - 按钮是一种命令组件，可发起一个即时操作。
- [全局配置 ConfigProvider](/components/config-provider/) - 在应用的最外层进行配置，一次设置，全局生效。一般用于设置国际化语言等功能。
- [链接 Link](/components/link/) - 链接的基本样式。
- [排版 Typography](/components/typography/) - 用于展示标题、段落、文本内容。

## 布局
- [分割线 Divider](/components/divider/) - 划分内容区域，对模块做分隔。
- [栅格 Grid](/components/grid/) - 栅格可以有效的保证页面的一致性、逻辑性、加强团队协作和统一。
- [布局 Layout](/components/layout/) - 页面的基础布局框架，常与组件嵌套使用，构建页面整体布局。
- [间距 Space](/components/space/) - 设置组件之间的间距

## 导航
- [面包屑 Breadcrumb](/components/breadcrumb/) - 面包屑是辅助导航模式，用于识别页面在层次结构内的位置，并根据需要向上返回。
- [下拉菜单 Dropdown](/components/dropdown/) - 页面上的命令过多时，可将备选命令收纳到向下展开的浮层容器中。
- [菜单 Menu](/components/menu/) - 收纳、排列并展示一系列选项的列表。
- [页头 PageHeader](/components/page-header/) - 页头位于页容器顶部，起到了内容概览和引导页级操作的作用。包括面包屑、标题等内容。
- [分页 Pagination](/components/pagination/) - 采用分页控制单页内的信息数量，也可进行页面跳转。
- [步骤条 Steps](/components/steps/) - 明示任务流程和当前完成程度，引导用户按照步骤完成任务。

## 数据展示
- [头像 Avatar](/components/avatar/) - 用作头像显示，可以为图片、图标或字符形式展示。
- [徽标数 Badge](/components/badge/) - 一般出现在图标或文字的右上角。提供及时、重要的信息提示。
- [日历 Calendar](/components/calendar/) - 日历组件。
- [卡片 Card](/components/card/) - 将信息分类后分标题、详情等区域聚合展现，一般作为简洁介绍或者信息的大盘和入口。
- [图片轮播 Carousel](/components/carousel/) - 用于展示多张图片、视频或内嵌框架等内容的循环播放，支持系统自动播放或用户手动切换。
- [折叠面板 Collapse](/components/collapse/) - 可以折叠 / 展开的内容区域。
- [评论 Comment](/components/comment/) - 展示评论信息
- [描述列表 Descriptions](/components/descriptions/) - 一般用于详情页的信息展示。
- [空状态 Empty](/components/empty/) - 指当前场景没有对应的数据内容，呈现出的一种状态。
- [图片 Image](/components/image/) - 展示和预览图片。
- [列表 List](/components/list/) - 最基础的列表展示，可承载文字、列表、图片、段落，常用于后台数据展示页面。
- [气泡卡片 Popover](/components/popover/) - 鼠标悬停、聚焦或点击在某个组件时，弹出的气泡式的卡片浮层。可以对卡片上的元素进行操作。
- [数值显示 Statistic](/components/statistic/) - 突出展示某个或某组数字、带描述的统计类数据。
- [表格 Table](/components/table/) - 用于数据收集展示、分析整理、操作处理。
- [标签页 Tabs](/components/tabs/) - 将内容组织同一视图中，一次可查看一个视图内容，查看其他内容可切换选项卡查看。
- [标签 Tag](/components/tag/) - 用于信息的选择、筛选、分类。用户通过标签进行信息反馈和交互操作。
- [时间轴 Timeline](/components/timeline/) - 按照时间顺序或倒序规则的展示信息内容。
- [文字气泡 Tooltip](/components/tooltip/) - 鼠标悬停、聚焦或点击在某个组件时，弹出的文字提示。
- [树 Tree](/components/tree/) - 对于文件夹、分类目录、组织架构等层级较多的内容，树可以清楚显示他们的层级关系，并具有展开、收起、选择等交互功能。

## 反馈
- [警告提示 Alert](/components/alert/) - 向用户显示警告的信息时，通过警告提示，展现需要关注的信息。
- [抽屉 Drawer](/components/drawer/) - 触发命令后，从屏幕一侧滑出的抽屉式的面板。
- [全局提示 Message](/components/message/) - 由用户的操作触发的轻量级全局反馈。
- [Modal 对话框](/components/modal/) - 在当前页面打开一个浮层，承载相关操作。
- [通知提醒框 Notification](/components/notification/) - 全局展示通知提醒，将信息及时有效的传达给用户。
- [气泡确认框 Popconfirm](/components/popconfirm/) - 点击元素，弹出气泡式的确认框。
- [进度条 Progress](/components/progress/) - 给予用户当前系统执行中任务运行状态的反馈，多用于运行一段时间的场景，有效减轻用户在等待中产生的焦虑感。
- [结果页 Result](/components/result/) - 用于反馈一系列操作任务的处理结果，当有重要操作需告知用户处理结果，且反馈内容较为复杂时使用。
- [骨架屏 Skeleton](/components/skeleton/) - 将加载中的数据用灰色占位。
- [加载中 Spin](/components/spin/) - 用于页面和区块的加载中状态 - 页面局部处于等待异步数据或正在渲染过程时，合适的加载动效会有效缓解用户的焦虑。

## 其他
- [固钉 Affix](/components/affix/) - 将页面元素钉在可视范围。当内容区域比较长，需要滚动页面时，固钉可以将内容固定在屏幕上。常用于侧边菜单和按钮组合。
- [锚点 Anchor](/components/anchor/) - 通过锚点可快速找到信息内容在当前页面的位置。
- [自动补全 AutoComplete](/components/auto-complete/) - 输入框的自动补全功能。
- [返回顶部 BackTop](/components/back-top/) - 可一键返回顶部的按钮。
- [级联选择 Cascader](/components/cascader/) - 指在选择器选项数量较多时，采用多级分类的方式将选项进行分隔。
- [复选框 Checkbox](/components/checkbox/) - 在一组数据中，用户可通过复选框选择一个或多个数据。
- [颜色选择器 ColorPicker](/components/color-picker/) - 用于选择和展示颜色
- [日期选择器 DatePicker](/components/date-picker/) - 选择日期。支持年、月、周、日类型，支持范围选择等。
- [表单 Form](/components/form/) - 具有数据收集、校验和提交功能的表单，包含复选框、单选框、输入框、下拉选择框等元素。
- [输入框 Input](/components/input/) - 基本表单组件，并在原生控件基础上进行了功能扩展，可以组合使用。
- [数字输入框 InputNumber](/components/input-number/) - 仅允许输入数字格式的输入框。
- [标签输入框 InputTag](/components/input-tag/) - 用来输入标签。
- [提及 Mention](/components/mention/) - 用于在输入中提及某人或某事，常用于发布、聊天或评论功能。
- [折叠列表 OverflowList](/components/overflow-list/)
- [单选框 Radio](/components/radio/) - 在一组相关且互斥数据中，用户仅能选择一个选项。
- [评分 Rate](/components/rate/) - 用于评分或打星的组件。
- [伸缩框 ResizeBox](/components/resize-box/) - 伸缩框组件。
- [滚动条 Scrollbar](/components/scrollbar/) - 用于替换浏览器默认滚动条样式。
- [选择器 Select](/components/select/) - 当用户需要从一组同类数据中选择一个或多个时，可以使用下拉选择器，点击后选择对应项。
- [滑动输入条 Slider](/components/slider/) - 滑动型输入器，展示当前值和可选范围。
- [面板分割 Split](/components/split/) - 将面板分割成两部分。
- [开关 Switch](/components/switch/) - 互斥性的操作控件，用户可打开或关闭某个功能。
- [文本域 Textarea](/components/textarea/) - 多行纯文本编辑控件，适用于评论或反馈表单中的一段意见。
- [时间选择器 TimePicker](/components/time-picker/) - 在弹出面板上选择时间，以便捷完成时间输入的控件。
- [数据穿梭框 Transfer](/components/transfer/) - 两栏布局的多选组件，将元素从一栏即时移到另一栏。
- [树选择 TreeSelect](/components/tree-select/) - 可以对树形结构数据进行选择。
- [触发器 Trigger](/components/trigger/) - 用于对元素添加 hover, click, focus 等事件，并且弹出下拉框。
- [上传 Upload](/components/upload/) - 用户可传输文件或提交相应的内容。
- [验证码输入 VerificationCode](/components/verification-code/) - 验证码输入组件
- [水印 Watermark](/components/watermark/) - 用于给页面的指定区域加上水印。
